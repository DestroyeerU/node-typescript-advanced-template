### Advanced template for Node + Typescript

### Features

- **Pre Code Made**
  - JWT Authentication
  - Yup validation
  - Files Upload

- **Code Formatter**
  - Eslint
  - Prettier
  - Root Import `~`

- **Tests with Jest**
  - User and Session CRUD tests
  - Auth unit tests
  - Factory with Faker

- **Prisma**
  - Env variables
  - User model example
  - File model for files upload
